MATLAB scripts used within W. Randall's MEng dissertation thesis.

1. 2nd Order ROM - Derived to model non-linear and snap-through behaviours in a clamped shallow arch. Uses a Lagrangian framework to derive Equations of Motion.
2. Co-Rotational FEM - Model non-linear snap-through behaviours in a clamped shallow arch. Uses formulations outlined in R. Wiebe's PhD thesis.
