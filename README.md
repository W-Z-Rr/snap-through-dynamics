Scripts used within W. Randall's MEng dissertation thesis.

1. 2nd Order ROM - Derived to model non-linear and snap-through behaviours in a clamped shallow arch. Uses a Lagrangian framework for Equations of Motion. - derived from: https://royalsocietypublishing.org/rspa/article/472/2190/20160172/57250/On-the-experimental-identification-of-unstable
2. Co-Rotational FEM - Models non-linear snap-through behaviours in a clamped shallow arch. Uses formulations outlined in R. Wiebe's PhD thesis. - https://dukespace.lib.duke.edu/items/819aaa89-24fb-4944-9759-5fb10ff5d347

Should operate within their own MATLAB instance; may need the signal processing toolbox to be installed.

3. Fusion 360 API Code - Uses the parameterised arch equation from: https://www-sciencedirect-com.sheffield.idm.oclc.org/science/article/abs/pii/S0020746216302037
4. Experimental Processing Code - Uses the STFT to evaluate the transient evolution of frequency-domain content. Experimental Data avaliable upon reasonable request.
